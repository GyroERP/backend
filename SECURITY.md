# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| main    | Yes       |

## Reporting a vulnerability

If you discover a security issue in GyroERP backend, please report it
responsibly. **Do not** open a public GitHub issue for security vulnerabilities.

Send details to:

**security@gyroerp.com**

Include:

- A description of the vulnerability
- Steps to reproduce
- Affected components or endpoints (if known)
- Any suggested fix (optional)

We aim to acknowledge reports within **5 business days** and will coordinate
disclosure with you before any public announcement.

## Scope

This policy applies to the GyroERP backend repository and its deployed API
services maintained by GyroERP.
